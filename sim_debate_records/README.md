# Simulation Debate Records

This folder stores the raw JSONL transcripts produced by the simulation pipeline. These files represent the actual turn-by-turn conversations generated by the LLM agents.

## Current Directory Layout

The folder contains subdirectories that hold the data used for the final project report, alongside top-level files from recent pipeline test runs:

- `gemma4_e4b/`
  - Final experiment runs using the aligned base Gemma model.
- `gemma4_uncensored/`
  - Final experiment runs for the aggressive uncensored Gemma variant.
- *Top-level `.jsonl` files*
  - Output from recent test executions of the `run_pipeline.ipynb` notebook.

## Where These Results Come From

The transcripts in this folder are written by the three simulation modules located in the `/simulations/` directory. They are typically executed via the `run_pipeline.ipynb` notebook.

1. **`simulations/naive_debate.py`**
   - Fixed-persona baseline debate.
   - Output pattern: `naive_{topic}_{debate_id}_{model}.jsonl`

2. **`simulations/reddit_aligned.py`**
   - Reddit-aligned debate using a real seed thread (`data/sim_seeds.jsonl`) and two aligned user profiles.
   - Output pattern: `reddit_{topic}_{submission_id}_{debate_id}_{model}.jsonl`

3. **`simulations/moderated_reddit.py`**
   - Reddit-aligned debate monitored by an active moderation agent.
   - Output pattern: `moderated_reddit_{topic}_{submission_id}_{debate_id}_{model}.jsonl`

## Experiment Matrix Details

The final experiment matrix in our paper compares two debate models:
- `gemma4:e4b`
- `fredrezones55/Gemma-4-Uncensored-HauhauCS-Aggressive:e4b`

Each run simulates 8 debate rounds. Because each round contains one generated reply from each of the two debate agents, a full transcript contains 16 generated debate turns (not counting the original Reddit seed context or inserted moderator interventions).

## JSONL Structure

### 1. Naive Debate Files
These files contain one generated debate turn per line. They do not include Reddit metadata or moderation scores.
- Typical fields: `debate_id`, `round_idx`, `agent`, `text`

### 2. Reddit-Aligned Debate Files
These files begin with the selected real Reddit seed context and then append the generated aligned replies. 
- Rows with `"generated": false` come from the original Reddit seed thread.
- Rows with `"generated": true` are the simulated LLM replies.
- Typical fields: `debate_id`, `submission_id`, `id`, `author`, `body`, `created_utc`, `round_idx`

### 3. Moderated Reddit-Aligned Files
These files follow the exact same Reddit-aligned structure, but include the Active Moderator's evaluation.
- Additional fields on generated turns: `toxicity`, `judge`, `issue_type`, `thread_cumulative_penalty`
- If moderation fires, the intervention turn is stored directly inside the transcript with `"type": "intervention"`, perfectly preserving the chronological flow of the mediated conversation.
