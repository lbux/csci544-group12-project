# Evaluations & Metrics (`/evaluations`)

This directory is responsible for grading the transcripts generated by the simulation pipeline and aggregating the results into our final report metrics.

## Code Flow

1. **`evaluate_debates.py`**: The Judge. Reads the JSONL transcripts from the simulation directory, reconstructs the conversational flow, and passes it to an LLM-as-a-Judge. The LLM uses a strict rubric to score the transcript on Alignment, Argument Quality, and Toxicity, saving a scorecard for each debate.
2. **`summarize.py`**: The Aggregator. Scans the generated scorecards, groups them by experimental condition (Naive, Reddit-Aligned, Moderated), calculates the mathematical averages, and prints the final formatted summary table.
